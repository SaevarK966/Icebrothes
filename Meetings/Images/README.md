# Meetings Images

Þessi mappa er fyrir myndir sem tengjast fundum Icebrothes.

Hér má setja inn:

- skjáskot frá fundum
- hópmyndir eftir fund
- myndir af fundarstað
- myndir sem eiga að birtast á `fundir.html`
- myndir sem eiga að fylgja einstökum fundargerðum

Dæmi um heppileg skráarnöfn:

- `2026-04-13-fundur-1.jpg`
- `2026-04-13-fundur-2.jpg`
- `2026-05-02-agency-fundur.jpg`

Þegar nýjar myndir eru komnar hingað get ég tengt þær inn á síðuna eða inn á ákveðna fundargerð.
