\# AGENTS.md



\## Repository purpose

This repository contains the Icebrothes website.



\## Before making changes

Always read `CREW\_CONTEXT.md` before editing copy, structure, styling, or visual identity.



\## Core rules

\- The correct crew name is \*\*Icebrothes\*\*.

\- Never rename it to \*\*Ice Brothers\*\*.

\- Preserve the crew's Icelandic identity and established lore.

\- Keep the site visually strong, dark, cinematic, and premium.

\- Avoid childish gaming-template aesthetics.

\- Use orange, black, white, and dark gray as the primary palette.

\- Keep the site static and compatible with GitHub Pages.

\- Use plain HTML, CSS, and lightweight JavaScript only.

\- Do not add frameworks or heavy dependencies unless explicitly requested.

\- Keep code clean, readable, and easy to edit later.

\- Prefer semantic HTML and responsive layouts.

\- Do not invent crew facts that are not supported by `CREW\_CONTEXT.md`.

\- If new references or assets are added, use them to improve the site while preserving the established identity.



\## Content guidance

\- Tone should feel confident, memorable, slightly aggressive, and cinematic.

\- Do not use generic AI marketing language.

\- Keep headings punchy and clean.

\- Keep body text short and readable.

\- Make the crew feel real and established.



\## Visual guidance

\- Strong hero section

\- Bold typography

\- Clean spacing

\- Card-based sections are welcome

\- Subtle hover effects and lightweight animations are welcome

\- Keep the design more tactical-premium than playful



\## Files

Main editable files are expected to include:

\- `index.html`

\- `style.css`

\- `script.js`
